# My First API

A simple REST API for user management on FastAPI.

## Technologies
- Python 3.8+
- FastAPI

### Clone the repository
```bash
git clone https://github.com/quinsaiz/my-first-api.git
cd my-first-api
```

### Create a virtual environment
```bash
python -m venv .venv
source .venv/bin/activate
```

### Install the requirements
```bash
pip install fastapi uvicorn
```

## Start the development server

```bash
uvicorn main:app --reload
```
